# Les opérateurs d'incrémentation

```php
$a = 0;
echo $a++; // Affiche : 1
echo $a; // Affiche : 1

$b = 1;
echo $b--; // Affiche : 0
echo $b; // Affiche : 0

$c = 0;
echo ++$c; // Affiche : 0, $c est modifié à l'instruction suivante
echo $c; // Affiche : 1

$d = 1;
echo --$d; // Affiche : 1 $d est modifié à l'instruction suivante
echo $d; // Affiche : 0
```
