# Les opérateurs de calcul

## Addition

```php
$a = $b + $c;
```

## Soustraction

```php
$a = $b - $c;
```

## Mutiplication

```php
$a = $b / $c;
```

## Division

```php
$a = $b + $c;
```

## Reste division euclidienne

```php
$a = $b % $c;
```

## La concaténation

```php
$a = $b . $c;

// Si $b = "Je suis "
// Et $c = "beau"
// Alors $a = "Je suis beau"
```
